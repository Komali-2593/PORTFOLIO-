# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built to showcase my skills, projects, certifications, achievements, and professional journey as a Computer Science student and aspiring Software Developer.

## 🚀 Live Demo

**Portfolio Website:** https://komali-2593.github.io/PORTFOLIO-

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Professional landing page with introduction
* About Me section
* Skills and Technology Stack showcase
* Projects portfolio section
* Certifications and achievements display
* Contact section with social links
* Clean and modern UI/UX
* Smooth navigation experience

## 💻 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Komali-2593/PORTFOLIO-.git
```

### Open the Project

```bash
cd PORTFOLIO-
```

Open `index.html` in your browser or use Live Server in VS Code.
